# PYtet
PYtet by Java by which you can PvP online. If I cannot accomplish it, maybe I will make it offline.   
就想用Java写个能线上对战的俄罗斯方块,如果写不出来线上对战的就写个本地双人的  
玩法/How to play:  
左移/Move Left: A  
右移/Move Right:D  
硬降/Hard Drop: W  
软降/Soft Drop: S  
换块/Hold:      Shift  
右旋/Clockwise: Right Button  
左旋/Counterclockwise:Left Button
